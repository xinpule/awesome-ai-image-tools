### ✂️ Isolate & Extract (Matting & Cleanup)
*Tools designed to separate subjects from their backgrounds, remove unwanted objects, and create clean, transparent assets.*

* **[Remove.photos](https://remove.photos)** - An incredibly fast, zero-click cloud AI tool that offers automatic background removal, precise object cleanup, and sky replacement right in the browser. Perfect for high-volume product photos.
* [Erase.bg](https://www.erase.bg/) - A smart AI background remover and image editor.
* [MagicEraser](https://magicstudio.com/magiceraser) - Remove unwanted things from images in seconds.
* [Slazzer](https://www.slazzer.com/) - Uses AI to remove or replace image backgrounds automatically.

### ✨ Enhance & Restore (Upscaling & Fixes)
*Solutions for breathing new life into low-resolution images, colorizing old photos, and sharpening details.*

* [Krea AI](https://www.krea.ai/) - Real-time AI generation and incredible image upscaling/enhancement.
* [VanceAI](https://vanceai.com/) - AI photo enhancement and editing tools suite.
* [Remini](https://remini.ai/) - Popular tool for unblurring, restoring, and enhancing image quality.
* [CodeFormer](https://github.com/sczhou/CodeFormer) - An open-source robust face restoration algorithm for old photos or AI-generated faces.

### 🎨 Generative AI & Synthesis
*Text-to-image and image-to-image models that create entirely new visual content.*

* [Leonardo.ai](https://leonardo.ai/) - Generate production-quality assets for your creative projects with pre-trained AI models.
* [Fooocus](https://github.com/lllyasviel/Fooocus) - An image generating software (based on Gradio) that focuses on prompting and generating, simplifying the Stable Diffusion workflow.
* [Civitai](https://civitai.com/) - The premier hub for discovering and sharing generative AI models, LoRAs, and embeddings.

### 🗜️ Optimization & File Processing
*Smart utilities for compressing, converting, and optimizing visual assets without losing perceptual quality.*

* [ImageOptim](https://imageoptim.com/) - Makes images load faster by removing bloated metadata and saving disk space.

### 🛠️ Front-End Previewers & Utilities
*Lightweight, elegant web tools for previewing, manipulating, and rendering complex file types (3D models, design files, docs) directly in the browser.*

* *(🚀 Submit your small and beautiful file preview/processing tools here via PR!)*
* [Photopea](https://www.photopea.com/) - Advanced online photo editor supporting PSD, XCF, Sketch, XD and CDR formats.
* [Gltf Viewer](https://gltf-viewer.donmccurdy.com/) - Drag-and-drop web tool for previewing glTF 2.0 3D models.

### 💻 Developer APIs & SDKs
*Integrate AI image processing directly into your own applications.*

* [Replicate](https://replicate.com/) - Run open-source machine learning models with a cloud API.
* [Hugging Face Inference API](https://huggingface.co/inference-api) - Access thousands of models via simple API calls.
* [Stability AI API](https://platform.stability.ai/) - Official API for Stable Diffusion models and generative workflows.

## 🤝 How to Contribute

We love community contributions! If you have built or discovered an amazing AI image tool, format converter, or web utility, please share it.

1. Fork this repository.
2. Add your tool to the most relevant category (Keep it alphabetical if possible).
3. Ensure the description is concise (1-2 sentences) and focuses on the core value.
4. Submit a Pull Request!
